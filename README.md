# Proyecto final de DAW - Supermercado online

En este repositorio se deja el código creado para desarrollar el supermercado online objeto de mi proyecto final para los estudios de Desarrollo de Aplicaciones Web realizados en el CIFP La Laboral de Gijón.

Se puede visitar la última versión disponible del proyecto en la siguiente URL:

- versión de producción: [mercatto-prod](https://mercatto.dev/)

Aunque la plataforma Vercel, lugar en el que está alojado el frontend del proyecto, provee dominios gratuitos, se ha preferido contratar un dominio de pago para la versión de producción.
